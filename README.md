# ZPRIA Account - Universal Identity for the Creative Ecosystem

## Overview
ZPRIA Account is a universal identity solution designed for the creative ecosystem. It provides secure authentication, personalized user experiences, and seamless integration with various services and platforms.

## Features
- Secure authentication with multi-factor authentication
- Theme customization with multiple color schemes
- Comprehensive user profile management
- Device management and security controls
- Payment method management
- Connected app integrations
- Notification preferences
- Activity logging and privacy controls

## Tech Stack
- React 19.2.4
- TypeScript
- Tailwind CSS
- React Router DOM
- Supabase for authentication and database
- Google Gemini AI integration

## Setup
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## Project Structure
- `/components` - Reusable UI components
- `/pages` - Application pages and views
- `/services` - API and service integrations
- `/SQL` - Database schema files

## Key Components
- Authentication flows (sign in, sign up, password reset)
- User dashboard and account management
- Security settings and privacy controls
- Theme personalization
- AI assistant integration
